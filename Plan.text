Assignment: Deep Neural Networks

------Objective:
- Implement baseline model (Logistic Regression from scratch)
  → Ek simple model jo sirf linear relation samajhta hai
- Implement MLP (Deep Neural Network from scratch)
  → Ek advanced model jisme layers hoti hain aur complex patterns seekhta hai
- Compare both models
  → Kaunsa better perform karta hai aur kyun

-----Dataset:
- Adult Income Dataset
- Source: Kaggle / UCI
- Problem Type: Binary Classification (income >50K or <=50K)
  → Output sirf 2 classes me hoga (0 ya 1)

------Steps:

1. Load Dataset
   - Read CSV file using pandas
   → Data ko table format me load karna (rows = samples, columns = features)

2. Data Preprocessing

WHY 2 libraries?

     → Jo data missing hai use remove ya fill karna

3. Baseline Model (Logistic Regression)
   - Implement Logistic Regression from scratch
